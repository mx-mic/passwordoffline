Program updates

Full list of updates and release notes

When I first released PasswordOffline, it was originally called DotPass. My intention back in 2021 was just to generate a password and that's it but now it's almost a full on GUI-based program!

My first commit was on 31/01/2021. This was when I added a README and license file. The license file is now gone, because my program is now under copyright law. You can view this commit here.

If you want to view the latest release, scroll down or click here.

First program release (out of beta, I deleted those releases for some reason, probabaly because they were tremendously buggy)

DotPass is now called PasswordOffline. Experience a new setup wizard, icon, and so much more! Now with less RAM needed for the program to run, it will perform operations faster than you can think! What are you waiting for? What's new in this release:
-New setup wizard
-Name changed
-New HD

Here are the release notes of version 1.1, which can also be viewed here.

Thank you so much for installing PasswordOffline! This release includes major bug fixes (A LOT!), more user-friendly UI elements and a better setup process! In this release we also made a portable version for people that don't like installing things or like to run things off removable media.
-Added full support for Windows Terminal
-Added support for Windows 11 including updated setup wizard to support rounded corners, new WinUI elements etc.
-Added more options during setup like opening HTML pages in end-user's default HTML file association
-Added more echo variables inside the code to help users get around the program easier
-A better title for everyone (especially those using Windows Terminal as the text got truncated in the last version)
-Major setup wizard bug fixes
-Major program bug fixes

Thank you so much for installing and updating PasswordOffline! Today's release includes more UI improvements, more setup options, and best of all, more generation options (including a very cool webpage you can go to by entering 5 after you have generated something!)

Here are the release notes of version 3.0, which can also be viewed here.

Listen, you lovely people! I started this project two years ago, just as a basic command-line file. Thanks to my increasing knowledge in the Windows codebase, PasswordOffline has evolved from that one tiny file, to a full-on program with a setup wizard, high-quality icons and better RAM usage! I want to thank you for sticking around here, and as the new year has approached I will try and start to release more versions and fixes. You are all lovely!
